### CBAM-TensorFlow2.0

- CBAM(Convolutional Block Attention Module) implementation on TensowFlow2.0
- 本论文来自ECCV2018，主要在传统的CNN上引入通道注意力机制和空间注意力机制提升性能。论文地址：[CBAM!](https://arxiv.org/abs/1807.06521)
